2 Prerequisites

  1. Install NPM.
  2. Unpack the binary in the directory, or install using bower:
    bower install --save protoboard
  3. Protoboard relies on HTML Import. So you will need an HTTP server serving static files. A simple one you can use is http-server: npm install http-server.
  4. Run http-server from the root directory of the project, and navigate to the file you want to open. A good one to start is ex/all/index.html.
