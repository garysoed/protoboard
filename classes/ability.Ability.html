ability.Ability

Base class for all abilities. Any abilities must extend this class and implement the methods in this class.

Constructor

ability.Ability(String name, trigger.Trigger defaultTrigger)

name

String

Name of the ability. This is used to identify an ability that has been added to an element. It therefore must be unique for the element.

defaultTrigger

trigger.Trigger

The default trigger to trigger this ability.

Properties

readonly String attrName

Name of the ability. This is also the name of the attribute on the element corresponding to this ability.
readonly String attrTrigger

The name of the attribute on the element corresponding to this ability that sets the trigger for this ability. This is the name of the ability with a -on postfix.

Methods

getTrigger(Element el) => trigger.Trigger | Null

Returns the trigger for this ability on this element.

el

Element

The element to return the trigger for this ability for.
return trigger.Trigger | Null

The trigger for this ability on this element, or null if there is none.
init(Element el)

Initializes the ability on the given element.

el

Triggers the effect of the ability on the given element.

el

The element to trigger the ability on.
updateConfig(Element el)

Updates the inner state of the ability based on changes to the element.
